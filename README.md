# rails-training

Ruby on Rails Getting Started.

### Install Rails

`gem install rails`

### Create Rails Application

`rails new blog`

# Develop on Existing Project

### Install Javascript Dependencies

`yarn install`

### Install Gems and Rails Dependencies

`bundle install`

### Make migrations

`rails db:migrate`

### Run Server and play with Rails!

`rails server`
